Move light:
Right arrow: light moves to the right.
Left arrow: light moves to the left.
Down arrow: light moves down the z axis.
Up arrow: light moves up the z axis.

Move camera:
D: move camera to the right.
A: move camera to the left.
W: move camera up the y axis.
D: move camera down the y axis.


Materials:
P: set material to gold
O: set material to silver
I: set material to plastic
U: set material human skin	 


Light: 
Z: set light red
X: set light green
C: set light white
V: set light white (but all light coefficients set to 1).
